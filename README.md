# Board

## 기술 스택
* IDE
```
 - IntelliJ
```
* Template Engine
```
 - Thymeleaf
```
* Back-end
```
 - Spring Boot
 - MyBatis
 - Lombok
```
* Database
```
 - MySQL
```
</br>

## 프로젝트 목적
* 웹 개발의 기본이 되는 게시판을 구현해 봄으로써 웹 서비스 구현에 대한 전반적인 과정을 경험한다.

* 테스트 케이스를 작성하고 테스트를 진행해 봄으로써 안전성과 효율성에 중점을 둔 서비스를 제작한다.

* MVC 패턴으로 소프트웨어를 개발하고 Model, View, Controller의 역할을 이해한다.
</br>

## 프로젝트 기능
* 게시글 등록, 조회, 수정 ,삭제 기능

* 댓글 등록, 조회, 수정 ,삭제 기능

* 게시글 검색 기능

* 파일 업로드, 다운로드 기능
</br>
